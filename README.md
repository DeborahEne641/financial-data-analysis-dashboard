📊 Financial Data Analysis Dashboard (Excel Project)

🔹 Overview

This project focuses on analyzing financial transaction data to uncover spending patterns, evaluate financial health, and provide actionable business recommendations.

The dataset was cleaned, transformed, and visualized using Microsoft Excel to build an interactive dashboard that supports data-driven decision-making.
🔹 Objectives

- Clean and prepare raw financial data
- Analyze income and expense patterns
- Identify high-cost areas
- Build a professional dashboard
- Provide actionable business recommendations

🔹 Dataset

- Over 2,000 transaction records
- Features include:
  - Transaction ID
  - Customer Name
  - Transaction Amount
  - Transaction Type (Credit/Debit)
  - Merchant
  - Location
  - Date
  - Account Balance

🔹 Data Cleaning

- Handled missing values (Customer, Merchant, etc.)
- Standardized inconsistent entries (e.g., NY → New York)
- Converted data types (text to numbers, formatted dates)
- Removed errors and corrected negative balances
- Cleaned text (uppercase, spacing issues)

🔹 Feature Engineering

New columns created:

- Transaction Category (Income vs Expense)
- Amount Category (Low, Medium, High)
- Month (for trend analysis)
- Balance Status (Low vs Healthy)

🔹 Dashboard

The dashboard includes:

- Total Transaction Value
- Total Income
- Total Expense
- Number of Transactions

Visualizations:

- Monthly Transaction Trend
- Income vs Expense Distribution
- Top Merchants by Spending
- Revenue by Location

🔹 Key Insights

- Expenses exceed income, indicating financial imbalance
- Spending is concentrated among a few merchants
- Certain locations contribute more to overall transactions
- Most transactions are low to medium value
- Some accounts show low balance risk

🔹 Business Decision

The company should implement a cost optimization strategy to reduce excessive spending and improve financial balance.

🔹 Implementation Plan

1. Identify high-cost merchants and locations
2. Reduce non-essential spending
3. Negotiate better pricing with vendors
4. Monitor monthly transaction trends
5. Balance income and expenses for sustainability
